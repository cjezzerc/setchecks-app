Welcome

This is an implementation of the application that was previously available as the NHS Value Set Management Tool (VSMT) Set Checks (Proof of Concept). With the closure of the VSMT project the Set Checks (Proof of Concept) is no longer available. This implementation is provided freely to the community.

The application includes 13 separate checks that run on user supplied SNOMED CT value sets to assess and provide feedback on set content. Different Set Checks run depending on whether the value set is being used for data entry or data extraction purposes, and whether the user wishes to check the value set against a single SNOMED CT release or between two SNOMED CT releases. The focus of these checks is for value sets that are used for data entry to, or data extraction from, electronic patient records. The checks indicate possible errors, content that might be missing, and how a set from an earlier UK SNOMED CT release should be changed due to new or changed content in a more recent UK SNOMED CT release.

There is a link on each page for feedback, including requests for help, demos or to generally contact me.

Using the Set Checks Application

All outputs should be reviewed and approved by relevantly qualified clinical professionals before changes are made to any value set being used to record or assess care data.

  • The input file must be in xlsx or tab-separated format with any number of columns, with a single header row, and a maximum of 5000 rows.
  • One column must contain SNOMED Identifiers; another column can optionally contain Terms; all other columns are disregarded by the set checks.
  • The selection of the Identifier and Term columns is made in the app after upload.
  • To upload your value set file, and start using the tool, click the big green "Browse for a file" button.
