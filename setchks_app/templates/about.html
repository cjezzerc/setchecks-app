About Value Set Checks (SNOMED CT)

This is an implementation of the Set Checks (Proof of Concept) application that was previously available to a restricted user base as part of the NHS Value Set Management Tool (VSMT) project.

  • The application includes twelve separate checks that run on user supplied SNOMED CT value sets to assess and provide feedback on set content.
  • The focus of these checks is for value sets that are used for data entry to, or data extraction from, electronic patient records.
  • Different Set Checks run depending on whether the value set is being used for data entry or data extraction purposes, and whether the user wishes to check the value set against a single SNOMED CT release or between two SNOMED CT releases.
  • The checks indicate possible errors, content that might be missing, and how a set from an earlier UK SNOMED CT release should be changed due to new or changed content in a more recent UK SNOMED CT release.

With the closure of the VSMT project the Set Checks (Proof of Concept) is no longer available.

This implementation is provided freely to the community and has no connection with NHS England.

The source code is available on GitHub.
